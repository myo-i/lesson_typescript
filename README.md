## npx tsc index.tsを実行したらindex.jsが生成された
TypeScriptはJavaScriptにコンパイルされるので.tsファイルが.jsファイルに変換されたものが生成される。  
なのでコンパイル後であれば、.tsファイルでエラーが起きていても実行できる。
